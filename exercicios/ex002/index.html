Parágrafos e Quebras de Linha


Você pode escrever um Parágrafo
de qualquer jeito.
Basta colocar tudo
no meio do par de tags p e /p.


Neste caso é mentira, pois para quebrar o texto em algum lugar específico
como esse, você precisa usar a tag <br>.


Não importa se damos ENTER num texto dentro de um código HTML pois não siginifa que a linha irá quebrar.


Simbolos e emojis

Para Adicionarmos simbolos e emojis usamos um conjunto de códigos chamado HTML Entities.

Esses códigos podem ser:


<
>
®
©...


Há também a possíbilidade de adicionarmos emojis em nossa página. Podemos usar uma enciclopédia online onde possuem o código do emoji e assim, podemos adicionar no nosso código HTML. Os emojis podem ser:
🤙
😄...


Dessa forma seu texto pode ficar diferente e divertido usando a forma correta de código html.
